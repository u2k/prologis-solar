# プロロジス 太陽光発電 ダッシュボード

全国31サイトの太陽光発電所における発電量データを集計・可視化するダッシュボード。

## 公開URL

GitHub Pages: `https://<your-username>.github.io/<repo-name>/`

## データソース

プロロジス所有の31サイト（関東16、関西7、中部2、東北3、九州3）における時間別発電量を、
2013年〜現在まで集計したものです。

## 表示内容

- 全国月別・年別発電量推移
- 地域別シェア（円グラフ）
- サイト別ランキング・詳細ビュー
- 累計発電量の規模感（物流センター・冷蔵倉庫・家庭世帯への換算）

## 更新頻度

毎日自動的に集計データが更新されます。
